Filter Search

How to use the Filter Search:

Use the drop-down menus, min/max parameters and keyword searches to create filters for your search. Use the CTRL (PC) or cmd (Mac) key to select more than one item from any drop-down menu. Check the box of every question for which you want to display the answer. The answer will be displayed for every country that meets your criteria.

If you want to display all the answers for a question without using the drop-down menu as a filter, check the box without selecting any item on the drop-down menu, or select the blank space at the top of the drop-down menu.
